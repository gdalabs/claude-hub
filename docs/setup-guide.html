What is claude-hub?

claude-hub is a local web dashboard for Claude Code. Sessions, chat history, and sub-agents are visualized in a browser UI. You can also chat with Claude directly from the dashboard.

claude-hub はブラウザベースの Claude Code ダッシュボード です。 セッション一覧、チャット履歴、サブエージェントの可視化、Claudeとの直接チャットが可能です。

Prerequisites / 事前準備

  • Node.js 18+ — node -v で確認
  • npm — Node.js に同梱
  • git — git --version で確認（推奨。なくても可）
  • Claude Code — インストール済みであること（claude --version）
  • Anthropic API Key — ダッシュボードからのチャット機能に必要
Tip: Node.js がまだ入っていない場合は brew install node（macOS）でインストールできます。
API Key: console.anthropic.com で取得できます。 セッション閲覧のみであればAPIキーは不要です。
Security: APIキーは事前に環境変数で設定するのがベストです。セットアップ時にターミナルに直接入力すると画面に表示されます：
export ANTHROPIC_API_KEY=sk-ant-xxxxx

Setup / セットアップ

ターミナルで1コマンド実行するだけです：

npx create-claude-hub my-dashboard

対話形式で以下のステップが自動で進みます：

1
Project Name / プロジェクト名
フォルダ名を入力（デフォルト: claude-hub）。引数で指定した場合はスキップ。
2
Download / ダウンロード
GitHub から最新版を自動ダウンロード（git clone、git がなければ tarball フォールバック）。
3
API Key / APIキー設定
Anthropic API キーを入力 → .env に保存。
環境変数 ANTHROPIC_API_KEY が設定済みなら自動検出。スキップも可。
4
Config / 設定ファイル作成
hub.config.json がテンプレートから自動生成。後からSettings UIで編集可能。
5
Install / 依存パッケージ
npm install を自動実行。macOS の permission エラーも自動対処。
6
Start / 起動
「Start now?」で Y → 開発サーバーが起動。ブラウザで http://localhost:5174 を開く。
Tip: 環境変数でAPIキーを事前に設定しておけば、入力プロンプトをスキップできます：
export ANTHROPIC_API_KEY=sk-ant-xxxxx && npx create-claude-hub my-dashboard

Settings Tab / 設定画面の使い方

ブラウザで http://localhost:5174 を開いたら、上部タブの Settings をクリック。 すべてUIから操作でき、JSON を直接編集する必要はありません。

1. Sites / サイトリンク

プロジェクトに紐づくデプロイ済みサイトのリンクを管理します。

ButtonAction
+ Add Group新しいプロジェクトグループを作成
+ Add Siteグループ内にサイトリンクを追加（emoji, 名前, URL）
×サイトやグループを削除

2. Quick Start / クイックスタート

よく使うコマンドをワンクリックでコピーできるショートカット集です。

ButtonAction
+ Add Section見出し（グルーピング用）を追加
+ Add Commandラベル + コマンドを追加
×項目を削除
Example: ラベル「My App」、コマンド cd ~/projects/my-app && claude と設定 → Quick Start からワンクリックでコピー → ターミナルに貼り付けて実行。

3. Discover Projects / プロジェクト自動検出

ローカルのフォルダをブラウズして、プロジェクトを自動検出します。

1
パスを入力
例: ~/projects
2
Browse をクリック
フォルダ一覧が表示。CLAUDE.md、package.json、.git のいずれかを含むフォルダがプロジェクトとしてハイライト。
3
+ Add で Quick Start に追加
cd /path/to/project && claude が自動生成されます。
4
Save で保存
設定が hub.config.json に保存され即座に反映。

Daily Usage / 毎日の使い方

Starting / 起動

cd my-dashboard && npm run dev

ブラウザで http://localhost:5174 を開く。

Stopping / 停止

ターミナルで Ctrl + C を押す。

Tips

Mobile: QR ボタンを押すと、同じ Wi-Fi 内のスマホからアクセスできる QR コードが表示されます。 ネットワークアクセスを有効にするには .env に HOST=0.0.0.0 を追加してください。
Theme: 右上の太陽/月アイコンでダーク/ライトテーマを切り替え。
Drag & Drop: サイドバーのプロジェクト順はドラッグで並び替え可能。

Security / セキュリティ

ItemDetail
API Server デフォルトで 127.0.0.1（localhost only）にバインド。外部からAPIにアクセスできません。
ネットワーク公開が必要な場合は HOST=0.0.0.0 を .env に追加。
Browse API ホームディレクトリ配下のみ閲覧可能。.ssh、.gnupg、.aws 等のセンシティブなディレクトリはブロック。
API Key .env ファイルに保存。.gitignore に含まれているため git にコミットされません。
Config PUT /api/config は構造バリデーション付き。不正なデータは拒否されます。
Important: APIキーを .env 以外の場所に書かないでください。コードやconfig にハードコードすると漏洩リスクがあります。

Troubleshooting / トラブルシューティング

ProblemSolution
npm install で permission error
npm install --cache /tmp/npm-cache
セッションが表示されない Claude Code を一度でも使ったことがあるか確認。~/.claude/ ディレクトリが必要です。
Chat でエラーが出る .env に ANTHROPIC_API_KEY が正しく設定されているか確認。
ポートが使用中
lsof -ti :5174 :3456 | xargs kill
ネットワーク経由でアクセスできない .env に HOST=0.0.0.0 を追加して再起動。

Manual Setup / 手動セットアップ

npx create-claude-hub が動かない場合は、手動でセットアップできます：

git clone --depth 1 https://github.com/gdalabs/claude-hub.git my-dashboard cd my-dashboard cp hub.config.example.json hub.config.json echo "ANTHROPIC_API_KEY=sk-ant-xxxxx" > .env npm install npm run dev
